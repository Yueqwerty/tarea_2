# Cuatro en Línea

Proyecto Cliente/Servidor para el juego Cuatro en Línea implementado en C++.


## Compilación

Para compilar el proyecto, ejecuta:

```bash
make
```

Esto generará los ejecutables en el directorio `bin/`.

## Ejecución

### Servidor

```bash
./bin/servidor <puerto>
```

### Cliente

```bash
./bin/cliente <IP_servidor> <puerto>
```
